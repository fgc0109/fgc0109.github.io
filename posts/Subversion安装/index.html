    Subversion安装

    September 28, 2014

    文章更新记录

    20190527:创建。

    一. Subversion 资料 #

    Subversion 官网地址：https://subversion.apache.org/

    Subversion 官网下载地址：https://subversion.apache.org/packages.html

    二. Subversion 安装 #

    安装 Subversion

    yum -y install subversion

    配置并启动 Subversion

    sudo gitlab-ctl reconfigure

    三. Subversion 管理 #

    1. 停止 Subversion # 

    killall svnserve

    2. 启动 Subversion #

    例如仓库地址为 /data/subversion/ 则对应命令如下:

    -d 表示守护进程, -r 表示在后台执行

    svnserve -d -r /data/subversion/dotbqp

    多个svn

    svnserve -d -r /usr/local/svn/svnrepos   (这里目录配到总目录，不是仓库目录)

在windows环境检出项目的时候，

    仓库1的检出地址为：svn://ip地址/warehouse1 仓库2的检出地址为：svn://ip地址/warehouse2

    3. 备份 Subversion #

    关闭所有运行的进程,并确认没有程序在访问存储库(如 httpd,svnserve 或本地用户在直接访问)

    压缩备份

    svnadmin dump /data/subversion/ | gzip > /data/subversion-backup.gz

    直接备份

    svnadmin dump /data/subversion/ > /data/subversion-backup.dump

    4. 还原 Subversion #

    新建一个版本库，用来存放要还原的内容：

    svnadmin create /data/subversion/dotbqp

    进行还原

    gzip -cd /data/subversion-backup.gz > /data/subversion-backup.dump
svnadmin load /data/subversion/dotbqp < /data/subversion-backup.dump


scp /data/subversion-backup.gz root@192.168.11.226:/data/subversion-backup.gz
scp root@192.168.0.5:/data/subversion-backup.gz /data/subversion-backup.gz

    检查版本库 svnadmin verify

    5. Nginx 反向代理SVN #

    安装配置apache

    yum install httpd

    安装插件 yum install mod_dav_svn -y

    systemctl enable httpd

    注意要把httpd的默认80端口换掉，避免与nginx冲突

    vim /etc/httpd/conf/httpd.conf

    LoadModule dav_module modules/mod_dav.so LoadModule dav_svn_module modules/mod_dav_svn.so

    Listen 8080

    设置版本库网站，要注意的是，版本库的网站不是iF.SVNAdmin的网站 vim /etc/httpd/conf.d/w_svn_8081.conf

    Listen 8081 DAV svn SVNParentPath /data/subversion AuthType Basic AuthName “svn repos” AuthzSVNAnonymous off AuthUserFile /data/subversion/dotbqp/conf/passwd AuthzSVNAccessFile /data/subversion/dotbqp/conf/authz Require valid-user

    访问路径 [host]:[port]/[repo]

    添加apache使用的密文密码

    htpasswd -nb fgc01 1EF%%pe#dwO*2i3Z htpasswd -nb ssq01 ZYoolPi$4Cw$nmx! htpasswd -nb zhm01 DPce$!bDBlDkwtNO
